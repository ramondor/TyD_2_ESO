Memoria Final de Proyecto

Tarea

Duración:
1 sesión
Agrupamiento:
Grupo taller (3/4 alumnos/as)

Por fín!!! Llegó el momento de presentar nuestro trabajo.

En grupo debemos completar una memoria de proyecto para acompañar con el objeto construido. En ella se recogerá toda la información de este; la propuesta, los bocetos, croquis, planos instrucciones de funcionamiento, etc,

Ánimo que ya acabamos. Debemos sentirnos satisfechos de todo lo trabajado y aprendido durante el curso. Haz que se note en tu trabajo final. Vamos!!!

Para facilitaros esta tarea os dejo unos ficheros adjuntos que podéis usar como base para vuestra memoria.

Rescata todos los documentos generados durante el proceso, sobre todo durante la planificación (Anteproyecto, proyecto final, bocetos, croquis, etc.) y utilízalos para este documento.

Haz fotos del proyecto terminado y del grupo junto al proyecto y añádelas para que quede más chulo.

Recuerda, además las indicaciones para realizar un buen documento de texto, que puedes encontrar en el apartado Anteproyecto del tema "Nuestro SUPERPROYECTO" del 1er TRIMESTRE
